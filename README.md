# world-news-api-clients - Docker Deployment

This repository contains the Docker deployment configuration for the **world-news-api-clients** MCP server.

## Google Artifact Registry

This image is automatically built and pushed to Google Artifact Registry using **Workload Identity Federation** (no JSON key required).

### Required Secrets
Add these secrets to your repository settings:
- `GCP_PROJECT_ID`: Your GCP project ID
- `GCP_PROJECT_NUMBER`: Your GCP project number (found in project settings)

### How It Works
GitHub Actions authenticates to GCP using OIDC tokens - no service account keys needed!
The bootstrap script sets up Workload Identity Federation automatically.

### Pull the Image
```bash
gcloud auth configure-docker us-central1-docker.pkg.dev
docker pull us-central1-docker.pkg.dev/YOUR_PROJECT/docker-images/news77:latest
```
